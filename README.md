frame
=====

Small personal collection of AMD based modules for rapid development. Since this is intended as personal framework
the API is ever changing and might be altered very quick. Trying to stick with [semver](http://semver.org/).

Dependencies
------------

The collection is dependent on [mout](http://moutjs.com/) and [cane](https://github.com/MathiasPaumgarten/cane).
